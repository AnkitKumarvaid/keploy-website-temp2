An Open-Source Functional Test Automation Tool

Keploy captures the network calls for an API request and replay them (including writes/mutations!) later during testing.

What Problems do Developers Face?

  • Test Scripts before every Release

  • Creating Testing Environment

  • Less than 50% Test Coverage

Auto-Mocked Dependencies

Keploy mocks any external service's response hence eliminating need for external dependencies.

Record-Reply Testing

Keploy compares test-cases generated from previously collected traffic against updated behavior of your application, and brings any differences to your attention.

Detailed Testing Coverage

Keploy tests the new version of application with thousands of test cases within minutes and generate instant reports.
